# Learning
## Learn to Learn

This repo gathers materials that students should find useful in learning how to learn.

## Unwillingness to Learn or to learn How to Learn.
It seems to me that the biggest obstacle to third-level students learning is not the technical complexity ofU the matters covered by undergraduate engineering programmes. The biggest obstacle, in my experience is the students' inability to learn at the level required to desrve a degree in engineering. Sometimes, in the more nexgative extreme, it is an unwillingness or a refusal to learn that hlds students back. More worryingly it is a refusal to learn how to learn.
